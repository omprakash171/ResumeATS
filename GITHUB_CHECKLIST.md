## Before Pushing to GitHub

1. Make sure `.gitignore` exists at project root
2. Verify `.env` file is NOT tracked (run: `git status`)
3. Create a GitHub repository
4. Add origin: `git remote add origin https://github.com/username/repo.git`
5. Push: `git push -u origin main`

## After Pushing

Tell users to:

1. Clone the repository
2. Copy `server/.env.example` to `server/.env`
3. Fill in their own credentials
4. Run `npm install` in both `server` and `client`
5. Start both dev servers
